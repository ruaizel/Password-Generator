🔐 Password Generator
A simple, interactive, and customizable Password Generator built using HTML, CSS, and Vanilla JavaScript.
This tool helps users create strong and secure passwords instantly.

🌟 Features
  🔢 Customizable Length
      Generate passwords with lengths ranging from 8 to 20 characters.
  🔤 Character Options
      Choose which characters to include:
      Uppercase (A–Z)
      Lowercase (a–z)
      Numbers (0–9)
      Symbols (!@#$%^&*)
      
  📊 Strength Indicator
      Displays real-time password strength:  
        - Weak
        - Medium
        - Strong
        
📋 One-Click Copy
Copy the generated password instantly to your clipboard.
  🚀 Getting Started
  📌 Prerequisites

No installations required! Just a web browser like:
  * Chrome
  * Firefox
  * Edge
  * Safari
  
⚙️ Installation & Usage
  Clone the repository
  Navigate to the project folder

▶️Run the project
  Open index.html in your browser
  Use the app
  Select password length
  Choose character types
  Click Generate Password
  Click Copy to save it
